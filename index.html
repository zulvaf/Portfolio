Complaint Management System

Desktop Application

Complaint Management System is a desktop application using Natural Languange Processing and Machine Learning to categorize whether tweets are complaints or not. It also determines the department related to complaint sentences in each tweet. Its aim is to help goverment of Bandung to process the complaints from people in Twitter without reading them one by one

Tools and Programming Languages: Java, Java Swing, NetBeans, Twitter API, InaNLP, Weka

Coneva Website

Website Application

Coneva is a travel website that help people to plan their own trip. User simply has to fill out the form that contains time period, accomodation preferences, transportation preferences, and tourist attractions, and then Coneva would plan the travel itinerary with lowest budget. Coneva would arrange the schedule, accomodation, and transportation for every trip. People can also choose a travel package that is already provided. Coneva is awarded as Top 10 Finalist in ICT Business Development Gemastik 9 2016

Tools and Programming Languages: HTML, CSS, Javascript, Bootstrap, Sublime Text

Dung Dung Local Search Engine

Website Application

Dung Dung is a local search engine developed in C# Programming Language. DungDung Local Search Engine can find all text files(.doc, .docx, .txt, .java, .cpp, etc), slides(.ppt, .pptx), and sheets (.xls, .xlsx) contained specific words in Windows Directory. DungDung Local Search Engine uses BFS and DFS Algorithm to locate the files, and PHP, HTML, and CSS to develop the front-end application

Tools and Programming Languages: C#, Visual Studio, PHP, HTML, CSS, Javascript, Sublime Text

eJob

Website Application

eJob is human resources management system in a form of a website. eJob helps human resource and hiring manager to allocate applicants to vacancy jobs. eJob can display all candidates who meet the specifications of each jobs. eJob can also show the statistics of jobs and applicants

Tools and Programming Languages: Laravel, PHP, CSS, Javascript, JQuery, MySQL, Boostrap, Sublime Text

Garong

Mobile Game

Garong is an endless arcade game developed with Unity Game Engine. The story tells about a burglar (Maling) who steal a hen and be caught by security (Hansip). The purpose of the game is to help the burglar to run from security while collecting coins to gain highest score. Garong can also be played in dual player mode using Wifi. In dual player mode, a player can choose to be a burglar or security, if the player is a security, the purpose is to catch burglar while collecting the coins

Tools and Programming Languages: C#, Unity, Mono Develop

Gomoku

Desktop Game

Gomoku is a strategy board game which allows several users in same connection to play it simultaneously. Every player must wait until their turn to place their pion on board. The first player who arranges their pions horizontally, vertically, or diagonally would be the winner. Gomoku is made based on Client-Server Communication Protocol and TCP Connection

Tools and Programming Languages: Python, Qt, Sublime Text

ITB Location Finder

Mobile Application

ITB Location Finder is mobile application to help user to find location in Bandung Institute of Technology by given latitude and longitude. This application also allows user to take pictures and then store them in the gallery. ITB Location Finder can be used by anyone who has a mobile with Android 3.0 minimum level of 11 (Honeycomb)

Tools and Programming Languages: Java, Android Studio, Google Maps API

Powerpuff Fish

Desktop Game

Powerpuff Fish is a simple game which allows its user to own virtual aquarium. User is able to add and feed the fishes, and the fishes will grow bigger if they have enough food or die otherwise. Powerpuff Fish is made using Object Oriented Programming Principle and Abstract Factory Design Pattern

Twiddle Twitter Analytics

Website Application

Twiddle is a web based twitter analytics that can classify tweets into several categories. User can input several keywords for each category and application would classify tweets based on those words. Twiddle is made with String Matching Algorithm such as Boyer Moore and Knuth-Morris-Pratt

Tools and Programming Languages: Java, JSP, HTML, CSS, Javascript, Twitter API, Sublime Text

We-KaWe

Desktop Application

We-KaWe is a classifier application similar to Weka, except it only implements KNN and Naive Bayes Algorithm. It can build and save the model from train data and classify test data based on the model. We-KaWe can also calculate total correctness and display the confusion matrix

Tools and Programming Languages: Java, Java Swing, NetBeans

ZIP SMS

Mobile Application

ZIP SMS is a SMS Messeger Application that can be used to send and receive encrypted message using block cipher. It also provides functionality to authenticate originality and integrity of a SMS using digital signature. The public and private key is generated using Elliptic Curve Cryptography. ZIP SMS provides safe and secure way to deliver SMS between two parties

Tools and Programming Languages: Java, Android Studio, SMS Manager
