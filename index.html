My Projects

Wordament 0


Wordament 0 is a simple replica of Wordament, word puzzle game published by Microsoft Studios in 2012, except it can only be played in command line, not mobile phone. Wordament 0 was developed in C Programming Language and using various Data Structures such as Stack, List, Queue, and Point.

DungDung Local Search Engine


DungDung is a local search engine developed in C# Programming Language. DungDung Local Search Engine can find all text files(.doc, .docx, .txt, .java, .cpp, etc), slides(.ppt, .pptx), and sheets (.xls, .xlsx) contained specific words in Windows Directory. DungDung Local Search Engine uses BFS and DFS Algorithm to locate the files, and PHP, HTML, and CSS to develop the front-end application

PowerPuff Fish


Powerpuff Fish is a simple game which allows its users to own virtual aquarium. Users are able to add and feed the fishes, and the fishes will grow bigger if they have enough food or die otherwise. This game is developed in Java Programming Language and Java Swing for Graphical User Interface. Powerpuff Fish is made using Object Oriented Programming Principle and Abstract Factory Design Pattern

Twiddle Twitter Analytics


Twiddle is a web based twitter analytics that can classify tweets into several categories. Twiddle is made with String Matching Algorithm such as Boyer Moore and Knuth-Morris-Pratt and developed in Java Programming Language as well as JSP, HTML, CSS, and Javascript

Simple Stack Exchange


Simple Stack Exchange is a website which allows its users to ask and answer questions, vote questions and answers, and comment the questions. Simple Stack Exchange is made based on REST Architecture and SOAP Protocol. This website is also implemented Web Security in form of authentication in Log In and Register Page. Simple Stack Exchange is developed in Java, JSP, HTML, CSS, and AngularJS

We-Kawe


We-Kawe is a classifier application similar to Weka, except it only implements KNN and Naive Bayes Algorithm. We-Kawe is developed in Java Programming Languange and using Java Swing for Graphical User Interface

Gomoku


Gomoku is a strategy board game which allows several users in same connection to play it simultaneously. Gomoku is made based on Client-Server Communication Protocol and using Python Programming Language as well as Qt Library for Graphical User Interface.
